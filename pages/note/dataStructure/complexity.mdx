# 复杂度计算

## 推导大 O 阶方法

1. 用常数 1 取代运行时间中的所有加法常数
2. 在修改后的运行次数函数中，只保留最高阶项
3. 如果最高阶项存在且不是 1，则去除与这个项相乘的常数

得到的结果就是大 O 阶。

下面是一个例子：

```js
int cal(int n) {
  int sum = 0;
  for (int i = 1;i <= n; ++i) {
    sum = sum + i;
  }
  return sum;
}
```

1. 用常数 1 取代运行时间中的所有加法常数

```js
int cal(int n) {
  int sum = 0; // c1
  for (int i = 1;i <= n; ++i) { // c2 * n
    sum = sum + i; // c3 * (n - 1)
  }
  return sum; // c4
}
```

2. 在修改后的运行次数函数中，只保留最高阶项

这里最高阶项是 n，所以保留 c2 * n

3. 如果最高阶项存在且不是 1，则去除与这个项相乘的常数

这里最高阶项是 n，所以去除 c2，得到 O(n)

## 常见的时间复杂度

- O(1)：常数复杂度
- O(logn)：对数复杂度
- O(n)：线性时间复杂度
- O(nlogn)：线性对数复杂度
- O(n^2)：平方复杂度
- O(n^3)：立方复杂度
- O(2^n)：指数复杂度
- O(n!)：阶乘复杂度
